Muestreo Sistem√°tico

(No es una muestra probabil√≠stica)

Recordar que la ventaja principal de una muestra probabil√≠stica es que nos permite usar teor√≠a estad√≠stica para hacer inferencia sobre la poblaci√≥n y con ello se tiene la capacidad de producir medidas de error y precisi√≥n.

Consiste en elegir una muestra en la cual solamente el primer elemento se toma al azar y el resto se selecciona sistem√°ticamente.

Puede ser combinado con otros procedimientos de muestreo, y puede ser utilizado en la practica por su relativa facilidad de administraci√≥n en el trabajo de campo.

Pasos para elegir una muestra sistem√°tica:

  1. definir la longitud del intervalo de muestreo

\(q = \frac{N}{n}\)

Si N no es divisible por n, se toma \(q\) como el entero inmediato anterior y habr√° muestras que sean de tama√±o n y otras de tama√±o n+1.

  1. Seleccionar \(q_0\) que sea un n√∫mero entero aleatorio entre 1 y \(q\).

  2. Seleccionar los siguientes elementos para que compongan la muestra

\(\{q_0, q_0 + q, q_0 + 2q, q_0 + 3q, \dots , q_0 + (n-1)q \}\)

ventajas

  • Es m√°s f√°cil seleccionar una muestra que en muestreo aleatorio.

  • Es factible producir una muestra representativa.

desventajas

  • Un mal arreglo en el marco puede producir muestras ineficientes.

  • No se pueden calcular estimadores de varianza con 1 sola muestra sistem√°tica.

El muestreo sistem√°tico puede llegar a ser utilizado para seleccionar unidades secundarias (USM).

Nota. En el pasado no se ten√≠a la facilidad computacional que se tiene ahora y se llegaba a trabajar tanto con tablas de probabilidad como con tablas de n√∫meros aleatorios. Es por ello que el uso de muestreo sistem√°tico pod√≠a ser una de las razones por las cuales era sencillo seleccionar una muestra de esa manera.

Nota. No hay expresiones v√°lidas para estimadores de varianza cuando se usa muestreo sistem√°tico, pero si los elementos est√°n ordenados, se pueden utilizar expresiones de muestreo aleatorio como una aproximaci√≥n y resulta un m√©todo conservador ya que las varianzas suelen ser menores.
